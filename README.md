# 🔧 MECHRA – AI Automotive Assistant

MECHRA adalah chatbot berbasis AI yang membantu pengguna mendiagnosis masalah kendaraan (motor & mobil) dengan cara yang sederhana dan mudah dipahami.

---

## 🌐 Live Demo

👉 https://mechra-ai-assistant.streamlit.app/

---

## 🚀 Features

- 💬 Chat-based interaction (seperti ChatGPT)
- 🧠 Context-aware response (menyesuaikan kondisi pengguna)
- ⚡ Smart suggestions (pilihan cepat tanpa banyak mengetik)
- 🔍 Internet search support (untuk informasi tambahan)
- ⚙️ Multi AI provider (Groq (LLaMA) & Gemini)
- 🧾 Notebook backend (penjelasan logika sistem)

---

## 🎯 Use Case

- Kendaraan tidak bisa start  
- Mesin mati di jalan  
- Bunyi aneh pada kendaraan  
- Masalah umum kendaraan lainnya  

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API (LLaMA)
- Google Gemini API
- Exa AI (search)
